pointInPolygon

(
  • pt
  • polygon
  • offset
)
Integer

Defined in common\jmUtils.js:671

判断点是否在多边形内 如果一个点在多边形内部，任意角度做射线肯定会与多边形要么有一个交点，要么有与多边形边界线重叠。 如果一个点在多边形外部，任意角度做射线要么与多边形有一个交点，要么有两个交点，要么没有交点，要么有与多边形边界线重叠。 利用上面的结论，我们只要判断这个点与多边形的交点个数，就可以判断出点与多边形的位置关系了。

Parameters:

  • pt Point

    坐标对象

  • polygon Array

    多边型角坐标对象数组

  • offset Number

    判断可偏移值

Returns:

Integer: 0= 不在图形内和线上，1=在边上，2=在图形内部
